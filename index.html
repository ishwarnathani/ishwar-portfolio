⚙ Hardware
Custom 8-bit & 16-bit CPU Design in Logisim
  • 8-bit SAP-1: 259 components, 629 wires, 13 subcircuits (ALU, PC, MAR, RAM, Registers, IR)
  • 16-bit CPU: 451 components, 1,685 wires — full fetch → decode → execute cycle
  • Both built entirely from logic gates — no higher-level abstractions
  • Verified arithmetic, logic, and memory operations across both datapaths
